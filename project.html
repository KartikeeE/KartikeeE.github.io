Long term determinants of health outcomes in India.

Home Portfolio Project

Aim

The project examines a combination of economic and infrastructural variables that determine India's health outcomes, as measured by life expectancy.
The project explores how improvements in social and economic determinants of health visibly correlate with increased life expectancy in India, using open-access data from the World Bank.

Data

All data was obtained from open and publicly available sources. The primary source of data was the World Bankâ€™s Development Indicators (WDI).

The indicators this project looks at are:

1. Health outcome: Life Expectancy at Birth (years)
2. GDP per capital (USD)
3. Health expenditure per capital (USD)
4. Universal Health Coverage Service Index (0-100)
5. Out of pocket health expenditure (% of current health expenditure)
6. Basic sanitation access (% of the population)

The data was accessed in two ways, direct API access and automated batch downloads. For the direct API access method, raw data was obtained from the WDI URLs and imported into a JSON file. It was then embedded into the Vega-Lite editor examples. The automated batch download method was used in Python to loop through indicators, combine datasets, and download and store the data in tidy, long-form.

Data cleaning and analysis

Frequent challenges faced in the project included identifying variables with consistent data in India, for example, inconsistent coverage across time frames, and large CSV files of raw data.
These challenges were addressed by choosing variables that were available and relevant to the topic, changing the examined time frames for all data for consistency, and cleaning the CSV data sets to keep only relevant data.
For data analysis, Python was used to clean, reshape, and standardise the data. Pandas was used to filter years, drop missing values, and convert the datasets to tidy, long-form data to make them consistent for use across charts.
Data visualisation was done using Vega-lite. Interactive elements were added to allow the viewer to examine changes in the data over time (life expectancy) and to compare different variables over time (life expectancy, GDP per capita, and health expenditure per capita). For charts such as life expectancy, the axes were adjusted to start at 50 years instead of 0 to ensure the data was well represented and the long-term trends were visible.

Conclusions

The overall long-term trends indicate that improvements in socio-economic factors that determine health improve health outcomes in India.
As GDP per capita and health expenditure per capita increased significantly, so did life expectancy.
Similarly, along with these variables, the Universal Health Coverage significantly increased, while out-of-pocket health expenditures dropped.
A final cross-country comparison shows where India stands in its relationship between life expectancy and GDP per capita.
