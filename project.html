Inflation and Cost of Living: India in International Context

PP434 Final Project — 5–8 visualisations (≤800 words)

Aims

This project examines how inflation has evolved in India compared with selected peer economies and what this implies for cost-of-living pressures and inflation volatility.

Data

Data are retrieved via the World Bank Indicators API using country codes and indicator codes. This approach supports replication and enables updates when the underlying source is refreshed.

Tools and Analytical Challenges

Charts are built in Vega-Lite and embedded using vegaEmbed. The workflow handles missing values through filtering and uses consistent transformations so time-series can be compared across countries.

Visualisations

Conclusions

Comparative evidence suggests inflation dynamics differ across economies in both level and volatility, motivating further work on drivers such as food prices, energy shocks, and exchange-rate pass-through.
