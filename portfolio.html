CC5: Scraper and API

Description:
The base URL used for the above chart is https://api.worldbank.org/v2/.
The elements used for the data are:
1. COUNTRY_CODE: ISO-3 country code (e.g., IND for India)
2. INDICATOR_CODE: World Bank indicator code (here: FP.CPI.TOTL.ZG = inflation, consumer prices, annual %)
3. Data format: format=json
The final URL is https://api.worldbank.org/v2/country/IND/indicator/FP.CPI.TOTL.ZG?format=json
